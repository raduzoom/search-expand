Search Expand Component - Flexible Usage Examples

1. Traditional JavaScript/TypeScript Usage

Import and use the component programmatically with flexible styling:

Change Skin: 

// JavaScript/TypeScript import
import { initSearchExpand, loadSkin } from 'search-expand';

// Initialize with default skin
const searchExpand = initSearchExpand(
    document.getElementById('traditional-example'),
    {
        viewSkin: 'default',
        persistentOptions: [
            { htmlContent: 'Apple', value: 'apple', currentStatus: 'unchecked' },
            { htmlContent: 'Orange', value: 'orange', currentStatus: 'checked' },
            { htmlContent: 'Banana', value: 'banana', currentStatus: 'unchecked' }
        ]
    }
);

// Change skin dynamically
searchExpand.setSkin('flat');

2. Web Component Usage

Use as a custom HTML element with attribute-based configuration:

Change Skin: 

<!-- HTML with data attributes -->
<dzs-search-expand
    data-skin="default"
    data-persistentOptions='[...]'>
</dzs-search-expand>

<!-- JavaScript to change skin dynamically -->
const component = document.querySelector('dzs-search-expand');
component.setSkin('flat');

3. UMD Module Usage (Browser)

Use via script tag in browser without build tools:

Change Skin: 

<!-- Include via script tag -->
<script src="dist-webpack/search-expand.js"></script>

<script>
// Access via global variable
const searchExpand = SearchExpand.initSearchExpand(
    document.getElementById('umd-example'),
    { viewSkin: 'default' }
);

// Change skin
searchExpand.setSkin('flat');
</script>

4. Custom Styling with CSS Variables

Override styles using CSS custom properties:


/* CSS Custom Properties for theming */
#custom-styles-example {
    --search-expand-primary-color: #e74c3c;
    --search-expand-border-radius: 20px;
    --search-expand-shadow: 0 4px 12px rgba(231, 76, 60, 0.3);
}
